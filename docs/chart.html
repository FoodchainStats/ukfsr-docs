Section 3 Chart Style Guide

This section will cover guidance on chart design and themes. For the text styleguide, see Chapter 2. There is general guidance on chart design produced by the Analysis Function.

3.1 Colours

The UKFSR uses the Analysis Function palette as recommended here. There is an R package, afcolours available on CRAN, which helps with implementing the palette.
